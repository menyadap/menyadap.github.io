### 3. Metode Cara Hack CCTV Lewat PC

#### **3.1 Download Angry Ip Scanner**

*Angry Ip scanner* termasuk **‘powerful GUI Port Scanner’** atau sangat ampuh dalam hal port scanning. Angry Ip Scanner tersedia untuk semua OS (Operasi Sistem).

#### **3.2 Tentukan Rentang Alamat Ip / Ip Range**

Penting bahwa bagaimana memilih rentang alamat ip yang tepat untuk bajak kamera CCTV. Kamera CCTV yang terhubung dengan koneksi internet broadband.

Jika menggunakan akses router broadband kemudian temukan alamat ip publik perangkat. Cukup ketik ***‘My IP’*** di Google atau Bing search bar. Google akan menampilkan alamat ip publik.

***Cara Bobol CCTV Private Mudah Terbaru***

Berikut 77.248.171.165 adalah IP publik. Jadi rentang IP bisa jadi 77.248.171.1 – 77.248.171.255 atau 77.248.171.1 – 77.248.175.255

#### **3.3 Konfigurasi Angry IP Scanner - Hack CCTV**

Pilih menu ***Tool*** greater than ***Preferences*** greater than ***Ports*** | tambahkan port *80,8080,23*.

Maka Port 80,8080 dan 23 ini yang akan di telusuri nantinya.

Kita perlu menambahkan fitur ***Web Detect***. Web detect dapat menampilkan rincian singkat tentang rincian perangkat yang terhubung ke internet. Seperti berikut :

* CCTV, camera model name
* CCTV, camera name
* Router name atau router model name

Untuk menambahkan web detect ikuti langkah berikut.

Pilih menu ***Tool*** > ***Fetchers*** > ***add*** (button) ***web detect*** > klik OK

#### **3.4 Scanning Rentang IP / IP range scanning**

Masukkan rentang ip di tab IP Range dan klik start.

Setelah proses scanning selesai akan menemukan informasi pada tab web detect misal seperti contoh sebagai berikut:

`RomPager / 4.07 UPnP / 1.0 ----- router`

`DVRDVS-Webs ----- CCTV camera`

`microhttpd ----- router`

`Jaring ----- CCTV camera`

#### **3.6 Crack Password Kamera CCTV - Hydra**

Jika dengan password default tidak berhasil maka perlu untuk melakukan crack / hack / peretasan. Hydra adalah *tool brute force* terbaik yang mampu untuk hack password kamera CCTV.

Sekarang jalankan Kali *(Kali Linux*), buka terminal dan ketik..

root@DeepHack:~# hydra -s 80 l admin -P /root/Desktop/wl/cctvpass.txt -e ns -t 16 *target IP* http*

Sintaks Hydra :

`-s 80 -- menentukan nomor port`

`-l admin - nama default login admin`

`-P /root/desktop/worldlists.txt - Pilih daftar kata Anda untuk brute force`

`-e --- password kosong`

`ns --- mencoba login dengan password dan tanpa password`

`http --- nama port untuk di attack`

Cara ini dapat hack password dan username kamera CCTV.
