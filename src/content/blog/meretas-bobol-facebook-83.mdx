## 15 Cara Hacker Hack Sosmed Orang

### 1. Hack Password - Phishing

Metode phising masih merupakan serangan yang paling populer digunakan untuk hack akun sosmed. Ada berbagai cara untuk melakukan serangan phishing.

Sederhana untuk trik phising ini, seorang hacker menciptakan sebuah halaman login palsu yang persis seperti halaman Facebook yang asli dan kemudian meminta korban untuk masuk ke halaman login palsu tersebut.

Begitu korban masuk, Email / User dan Password sosial media korban disimpan ke dalam file teks, dan si hacker kemudian mendownload file teks tersebut dan mendapatkan alamat email dan sandi korbannya.

#### **Metode Menghindari Serangan Phishing:**

* Jangan Login akun sosial media di perangkat lain
* Gunakan Chrome, karena browser ini dapat mengidentifikasi laman phishing.
* Hindari email yang meminta untuk login di akun sosial media, pada halaman yang tidak diketahui.

### 2. Hack Password Dengan Keylogging

**Keylogging** adalah cara mudah untuk hack password Sosmed facebook, instagram, telegram dll. Keylogging kadang bisa sangat berbahaya bahkan orang yang memiliki pengetahuan komputer yang baik bisa terjebak.

Keylogger pada dasarnya adalah sebuah program software, sekali dipasang di komputer korban, akan merekam semua aktifitas korban dengan komputernya.

Log ini kemudian dikirim kembali ke penyerang baik melalui FTP atau langsung ke alamat email hacker.

Keylogger sendiri merupakan aplikasi hack akun sosial media atau software hack sosmed yang sangat ampuh dan terkenal.

#### **Cara Menghindari Keylogger:**

* Selalu download software dari situs terpercaya
* Scan USB drive untuk menghindari virus
* Instal antivirus

### 3. Password yang Tersimpan pada Browser

Seperti yang kita tahu, setiap kali kita login sosial media pada Browser, browser meminta pengguna untuk menyimpan username dan password di komputer.

Maka siapa pun bisa hack akun Sosmed melalui pengelola **password pada browser**.

Anda bisa mengunjungi URL dibawah ini dan lihat semua username dan password yang anda simpan di browser.

`chrome://settings/passwords`

#### **Melindungi Akun Sosial Media Anda:**

* Jangan pernah menyimpan data login pada browser, apalagi di warnet.
* Selalu gunakan kata sandi yang kuat pada komputer Anda.

### 4. Hack Sosmed Dengan Sessiion Hijacking

**Hack Sessions** (***HTTP Session Hijacking***) sering kali sangat berbahaya jika mengakses sosial media menggunakan koneksi HTTP (tidak aman).

Dalam ***Session Hijacking Attack***, seorang hacker mencuri cookie browser target yang digunakan untuk autentikasi pengguna di situs web dan menggunakannya untuk mengakses akun korban.

Session hijacking biasanya sering digunakan pada koneksi LAN dan WiFi.

### 5. Sidejacking With Firesheep

**Sidejacking Attack** sering terjadi pada akhir tahun 2010, namun masih populer sampai sekarang. Firesheep digunakan untuk melakukan sidejacking Attack.

Firesheep hanya berfungsi saat penyerang dan korban berada pada jaringan WiFi yang sama saja tidak bisa digunakan selain itu.

Serangan sidejacking pada dasarnya adalah nama lain dari HTTP Session Hijacking, namun lebih ditujukan pada pengguna WiFi.

#### **Menghindari Session Hijacking Attack:**

* Hindari kebocoran cookies via HTTP
* Log off website ketika sudah selesai
* Hindari memakai WiFi Public
* Gunakan VPN

### 6. Mobile Phone Hacking

Jutaan pengguna sosial media mengakses Sosmed melalui HP mereka. Jika hacker bisa mengakses ponsel korban maka dia mungkin bisa mengakses akun sosial media.

Ada banyak aplikasi Mobile Spying yang digunakan untuk memantau aktivitas HP korban. Aplikasi Spy (mata-mata) yang paling populer adalah **Mobile Spy** dan **Spy Phone Gold**.

#### **Menghindari Mobile Spying:**

* Gunakan aplikasi antivirus yang terpercaya pada smartphone.
* Jangan pernah memasang aplikasi dari sumber yang tidak dikenal, diluar Play Store.
* Uninstall aplikasi mencurigakan, tidak dikenal.

### 7. DNS Spoofing

Jika target dan penyerang berada di jaringan yang sama, penyerang dapat menggunakan metode **DNS spoofing attack** ini dan mengubah halaman sosial media asli ke halaman palsu dan karenanya bisa mendapatkan akses ke akun sosial media seseorang.

#### **Agar Terhindar Dari DNS Spoofing:**

* Konfigurasi agar keamanan mungkin terhadap cache poisoning.
* Kelola server DNS Anda dengan sangat aman.
* Pisahkan authoritative function dengan resolving function menggunakan server yang berbeda.

### 8. USB Hacking

Jika penyerang memiliki akses secara langsung ke komputer, dia bisa saja memasukkan **USB hack** yang sudah diprogram dengan sebuah fungsi untuk secara otomatis mengambil dan mengekstrak kata sandi tersimpan di browser Internet dengna mudah.

#### **Agar Terhindar dari USB Hacking:**

* Masukkan perangkat USB yang terpercaya ke komputer.
* Jangan membeli USB bekas ***(rekomendasi baru)***.
* Scan USB setelah tersambung pada komputer.
