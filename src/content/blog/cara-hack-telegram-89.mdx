## 2. Cara Sadap Telegram - Versi Web

Untuk memudahkan penggunaan maka pihak Telegram menghadirkan *Telegram Web* seperti yang satu ini. Dengan Telegram Web ini nantinya akan lebih mempermudah bagi yang ingin menggunakan telegram di komputer atau PC.

Seperti yang diketahui bahwa selama ini masih banyak yang beranggapan bahwa telegram hanya bisa digunakan pada ponsel saja. Namun dengan menggunakan Telegram Web ini maka bisa mengoperasikan Telegram di berbagai perangkat dengan Browser.

Hanya saja, umumnya banyak orang yang tidak tahu bahwa ternyata Telegram juga tersedia versi Web ini bisa dimanfaatkan untuk melakukan hack Telegram.

Cara untuk melakukannya sendiri tergolong sederhana dan mudah untuk dilakukan siapapun dan kapanpun.

Langkah pertamanya adalah dengan menyuruh target untuk masuk ke Telegram Web menggunakan akun Telegram target. Setelah berhasil masuk, bisa memberikan berbagai alasan untuk tidak melakukan logout.

Jika si target mau untuk tidak logout maka sudah berhasil mengambil alih akun Telegram. Jadi nantinya apapun aktivitas si target ketika menggunakan akun Telegram maka pengguna bisa memantaunya menggunakan Telegram Web.

Namun usahakan untuk selalu berhati-hati supaya tidak diketahui atau disadari oleh si target bahwa telegram dia sedang diawasi.
