## Alat Wajib Seorang Hacker! 5 Fakta Aplikasi Termux

Termux sendiri merupakan sebuah terminal emulator android yang memiliki situs website resmi [termux.com](https://termux.dev/en/). Selain itu pula aplikasi Termux ini memiliki 5 fakta yang menarik seperti berikut ini.

## 1. Termux Pemasangan Mudah

Jadi memasang dan menjalankan aplikasi command line Termux di smartphone kamu tak perlu lagi setting tambahan atau harus melakukan rooting Android.

Nah kemudahan lainnya ialah ukuran aplikasi ini tak sebesar fungsinya yakni hanya berukuran ***235Kb***, ya mungkin ini lebih kecil dari ukuran 1 foto yang tersimpan pada galeri ponsel.

## 2. Termux Alat Para Hacker Professional

Aplikasi ini telah memiliki lebih dari 5 juta kali download pada toko aplikasi resmi *Play Store (Android)* karena memang aplikasi ini banyak digunakan selain hacker pemula juga hacker professional atau yang sudah ahli untuk menjalankan sebuah program.

## 3. Tutorial Termux Banyak Tersedia

Tak perlu ragu-ragu untuk mencari tahu bagaimana cara untuk menggunakan aplikasi ini untuk belajar menjadi seorang hacker profesional.

Karena aplikasi ini memiliki banyak tutorial yang bisa kamu akses di artikel Google Search, Youtube, bahkan aplikasi sosial media seperti Facebook.

Membuat tutorial menggunakan aplikasi Termux menjadi bahan konten penulis maupun konten creator Youtube karena bisa meningkatkan jumlah pengunjung website atau channel karena banyaknya pengguna Termux tidak tahu bagaimana cara menggunakannya.

## 4. Termux dan Fungsi Tambahan

Selain tujuan utamanya sering digunakan untuk kegiatan hacker, ternyata aplikasi Termux juga memiliki fungsi tambahan yang didapatkan dari fitur atau ***package*** yang tersedia dalam aplikasi tersebut.

Fitur atau package yang kami maksud seperti berikut ini

* ***Tree***: Fitur ini bisa kamu gunakan sebagai media untuk membantu kamu untuk mencari file yang tersembunyi dalam penyimpanan file manager.
* ***Unrar***: Pusing dengan iklan yang terus muncul pada ponsel cerdas? gunakan saja fitur ***Unrar*** pada aplikasi Termux ini untuk block semua iklan yang kerap dianggap mengganggu aktivitas dalam aplikasi.
* ***AG***: fitur selanjutnya ini berfungsi untuk membantu kamu menemukan file dalam file manager meskipun dengan cepat meskipun banyak sekali sub folder yang ada pada ponsel. Lalu apa bedanya fitur ini dengan fitur Tree sebelumnya? bedanya ialah fitur atau ***package AG*** dapat mencari file atau frasa kalimat sekalipun pada suatu file.

## 5. Sebagai Tes Keamanan Siber

Untuk yang sudah ahli, biasanya hacker yang sudah professional menguji akun mereka menggunakan aplikasi Termux, untuk membuktikan seberapa kuat keamanan mereka bisa bertahan dengan aplikasi bash / command line pada Android.
