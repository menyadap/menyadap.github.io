## Cara Install Kali Nethunter di Termux Android

Cara Install Kali Nethunter di Termux

Download Termux: https://play.google.com/store/apps/details?id=com.termux&hl=en&gl=US

1. Pastikan kalau sudah pasang aplikasi Termux di hp android, jika belum download terlebih dulu aplikasi Termux melalui link yang tersedia dan install.
2. Hubungkan koneksi internet di hp android, baik data seluler atau jaringan WiFi.
3. Buka aplikasi Termux dan langsung masukan baris perintah di bawah ini untuk update dan install curl:

`apt update && apt upgrade –y`

`apt install curl`

4. Kemudian pastikan kalau sekarang berada di direktori Home dengan masukan baris perintah:

`Cd $HOME`

5. Masukan baris perintah berikut ini:

`curl –LO https://raw.githubusercontent.com/Hax4us/Nethunter-In-Termux/master/kalinethunter`

6. Selanjutnya, kamu perlu ubah persetujuan / permission file ***kali nethunter*** dan gunakan kode perintah di bawah ini:

`Chmod +x kalinethunter`

`./kalinethunter`

7. Sekarang tinggal tunggu saja beberapa saat hingga proses download selesai dilanjutkan dengan menjalankan baris perintah di bawah ini:

`startkali -r`

8. Download dan pasang *kali archive keyring* dengan menggunakan beris perintah berikut ini:

wget https://http.kali.org/pool/main/k/kali-archive-keyring/kali-archive-keyring-udeb_2024.1_all.udeb

`apt install ./kali-archive-keyring_2020.2_all.deb`

9. Kemudian, tahap ini untuk update repo dengan beris perintah berikut:

`apt update`

10. Selesai, kamu sudah berhasil untuk install ***kali nethunter*** di aplikasi Termux yang ada di hp android.
