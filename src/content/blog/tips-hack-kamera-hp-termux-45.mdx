## **Cara Hack Kamera HP - Termux**

Berikut ini cara sadap kamera HP menggunakan script termux remote kamera terbaru. Jika kalian belum download dan install termux maka download termux terlebih dahulu disini, Termux Apk.

### 1. Install Script HackCam Termux

Langkah pertama yang bisa dilakukan untuk hack kamera hp orang lain adalah dengan menginstall script hack kamera ***Termux Spy*** seperti yang satu ini.

Soalnya pengguna tidak akan bisa melakukan penyadapan pada ponsel kamera si target tanpa terlebih dulu menginstall script yang satu ini.

Bagi yang belum tahu, maka nantinya pengguna mesti masukan perintah seperti di bawah ini:

$apt update && apt upgrade

$pkg install openssh

$pkg install git

$pkg install mc

$git clone  https://github.com/BHC-FAKE/HackCam

$cd HackCam

$ls

$sh install.sh

Nantinya tinggal menunggu proses install ini sampai selesai.

### 2. Jalankan Script Hack Kamera HP

Apabila proses install seperti di atas sudah selesai dilakukan maka yang perlu dilakukan selanjutnya adalah dengan memulai menjalankan script hack kamera ponsel.

Hal ini berguna untuk membuat proses penyadapan kamera ponsel ini bisa berjalan sesuai dengan keinginan.

Bagi pengguna yang belum tahu, command yang mesti di masukan untuk menjalankan script hack kamera Android adalah seperti berikut ini:

`$ls`

`$bash HackCam.sh`

Apabila pengguna sudah melakukannya dengan tepat maka tidak berlangsung lama akan segera muncul tampilan awal script kamera ***saycheese***.

Tampilan awal script kamera ini sendiri juga menjadi pertanda bahwa sejauh ini pengguna sudah melakukan langkah-langkah dengan tepat.

Soalnya jika sudah muncul tanda yang satu ini maka pengguna sudah bisa melangkah ke tahap yang berikutnya.

### 3. Mulai Lakukan Sadap Kamera

Nantinya akan muncul tampilan yang membuat memilih di antara dua pilihan. Dari dua pilihan tersebut maka sebaiknya menjatuhkan pilihan pada server.net saja yang lebih memudahkan.

Apabila sudah, tinggal masukan ***command*** 01 pada server tersebut. Tidak membutuhkan waktu lama akan muncul tampilan khusus yang merupakan subdomain.

### 4. Bagikan Link pada Target

Langkah terakhir yang bisa dilakukan adalah membagikan link pada target. Untuk link yang dimaksud sendiri adalah link pada tampilan subdomain seperti di atas barusan. Tugas pengguna adalah membagikan link pada si target tanpa curiga.

Apabila si target mau klik link tersebut maka tidak membutuhkan waktu lama maka ponsel kamera milik si target sudah berhasil di hack dengan mudah.
