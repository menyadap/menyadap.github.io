## Cek Pesan Messenger yang Sudah Dihapus

Cara Melihat Pesan Messenger yang Sudah Dihapus

Kami memiliki 4 cara yang dapat di praktikan untuk mengetahui isi pesan yang telah dihapus di Messenger berikut ini:

### 1. Melihat Pesan Messenger Terhapus Lewat PC

Metode ini bisa Anda lakukan di perangkat laptop maupun komputer dengan bantuan aplikasi browser. Berikut langkah-langkahnya:

1. Pertama, silahkan buka browser seperti FireFox, Chrome, atau lainnya
2. Masuk ke situs Facebook
3. Login ke akun Anda
4. Buka menu Pengaturan di akun Anda
5. Masuk opsi menu, *your Facebook Information*
6. Anda akan melihat sejumlah informasi
7. Tekan pilihan *Download Your Information*
8. Checklist bagian Message dan klik Create File
9. Proses akan berjalan kurang lebih 30 menit
10. Jika sudah selesai, sekarang klik Download untuk menyimpan filenya
11. Buka file yang sudah berhasil di download tadi
12. Jika format ZIP atau RAR, perlu ekstrak terlebih dahulu baru melihat isi filenya
13. Selesai

Selain menggunakan komputer atau laptop, metode ini juga bisa di praktik kan pada smartphone melalui aplikasi browser apa saja.

Langkah-langkahnya pun hampir sama tetapi dengan syarat Anda harus login ke akun Anda pada situs Facebook menggunakan browser yang ada.

### 2. Cara Cek Pesan Messenger yang Sudah Dihapus di HP Android

Bagi Anda pengguna HP Android dan ingin tahu Cara Melihat Pesan Messenger yang Dihapus di HP Android bisa gunakan cara di bawah ini.

1. Pastikan sudah memiliki aplikasi Facebook di HP Android
2. Buka aplikasinya dan login ke akun Anda
3. Buka menu Pengaturan lalu pilih Akses Informasi Anda
4. Uncheck list yang menurut Anda tidak perlu
5. Checklist bagian Pesan
6. Tap Buat file dan tunggu hingga mendapatkan pemberitahuan lanjutan
7. Jika sudah, sekarang download file tersebut
8. Ekstrak terlebih dahulu untuk mengetahui isi filenya
9. Selesai

Anda juga bisa backup pesan Messenger supaya isi pesan sebelumnya bisa dilihat kembali di kemudian hari.

Backup pesan setiap 1 minggu atau 2 minggu sekali supaya pemulihan pesan nantinya bisa dilakukan dengan mudah.

### 3. Cara Melihat Pesan Facebook Messenger Dengan ES File Explorer

Pada cara ketiga ini, untuk sadap messenger yang sudah dihapus, Anda memerlukan aplikasi tambahan yang diperlukan bernama ES File Explorer untuk membantu mengembalikan pesan yang sudah dihapus sebelumnya.

Untuk melakukan cara ini, pastikan terlebih dahulu kalau Messenger belum dinonaktifkan sehingga bisa mengakses.

1. Langkah pertama yang harus dilakukan Anda menginstall aplikasi ES File Explorer  [Disini](https://trikwiki.com/excellent-ideas-and-examples-of-saas-startups/?url=aHR0cHM6Ly9lcy1maWxlLWV4cGxvcmVyLmlkLnVwdG9kb3duLmNvbS9hbmRyb2lk)
2. Jika aplikasi sudah terpasang, langsung buka saja ***ES File Explorer***
3. Sekarang buka penyimpanan Telepon karena di sinilah berbagai file dan data aplikasi tersimpan
4. Buka folder ***data/com.facebook.orca/cache***
5. Buka file fb_temp
6. File tersebut berisi backup dari akun Facebook

Jika file sebelumnya berupa gambar, maka suruh pengirim pesan untuk mengirim kembali file gambar yang dihapus sebelumnya.
